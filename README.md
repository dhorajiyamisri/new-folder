📊 Smart City Energy Consumption Dashboard

📌 Project Overview
This project analyzes energy consumption data of a smart city to identify usage patterns, peak demand, outages, and inefficiencies. The analysis is performed using SQL for data querying and Power BI for visualization.

🎯 Objectives
Analyze energy consumption across different zones
Identify high energy usage sectors
Monitor peak usage patterns
Detect faulty meters and outages
Provide insights and recommendations for improvement

📂 Dataset
File Name: SmartCityEnergy.csv
Type: Synthetic dataset generated using AI tools

Contains:
MeterID
Zone (North, South, Central)
ConsumerType (Residential, Commercial, Industrial)
Date
EnergyConsumed_kWh
PeakUsage_kW
OutageMinutes
MeterStatus
TariffRate

🛠 Tools & Technologies Used
MySQL (Database & SQL Queries)
Power BI (Dashboard & Visualization)
Excel (Data Handling & Output Storage)

🧮 SQL Analysis Performed
Zone-wise total and average energy consumption
Top 5 highest energy consumers
Monthly energy consumption trends
Average cost per zone
Identification of faulty meters
Energy efficiency analysis (usage vs outage)
Peak usage comparison (weekday vs weekend)

📊 Dashboard Features
KPI Cards:
Total Energy Consumption
Average Peak Usage
Total Outage
Total Cost

Visualizations:
Bar Chart (Zone-wise Consumption)
Line Chart (Monthly Trend)
Pie Chart (Consumer Type Distribution)

Filters (Slicers):
Zone
Consumer Type
Date

🔍 Key Insights
Central zone shows the highest energy consumption
Industrial sector consumes the most electricity
Peak usage is higher on weekdays
Some zones experience higher outage durations
Faulty meters are present and affect monitoring
